Values Menu - tool to select multiple values for multiple attributes and insert them in a cell with a separator that you choose;

String Tools:

  • Remove Diacritics - range based - Select a range wnd it will attempt to replace diacritic characters with normalized ones; - Warning: slow
    • Use Catalog URL Converter as a faster replacement;
  • Remove in-string duplicates - range based - splits a string by "-" then removes all duplicate strings found, joins them back by "-"; - Warning: slow 
  • Catalog URL Converter - formula based - replaces all given characters from a string witht the key, value paris found in a character map, values being set in unicode. Almost entirely used for URL normalization since it replaces diacritics and cyrillic/greek characters;.
  • Change Case - range based - multiple options to change the values found in the cells from the selected range to different character cases; - Warning: slow

Exporting:

  • SaveAsCSV - action based - Saves all sheets within the Spreadsheet into a new folder in drive..

Other Utilities:

  • Get sheet names - formula based - It generates a list of all the sheets found in the current active spreadhseet;
  • Extract sitemap URLs V1 - formula based - parses XML and extracts <loc> tags;
  • Extract sitemap URLs V2 - formula based - more advanced parsing with namespace given. Provides same functionality as V1;
  • HTTPResponse - formula based - return HTTP response of given URL;
  • Generate Named Ranges from Mapping - action based - given a map of names, ranges in Apps Scripts it automatically populates the Spreasheet with NamedRanges;
  • Remove All Named Ranges - action based - useful to use paired with named range generator;
  • addAttributeValidation - single cell based - if applied on a cell it grabs the value from the right adjacent cell and pases it as a Values from Range Datavalidation.
    • requires that the adjcent cells' value is a valid NamedRange within the spreadsheet;
  • Set Permission - App Script use only - grants a given e-mail address full permissions across all sheets;
